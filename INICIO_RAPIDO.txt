## 🛑 PROBLEMAS COMUNES Y SOLUCIONES

### "Kivy no encontrado"
```bash
pip install kivy kivymd --upgrade
```

### "Puerto 5000 ya en uso"
Cambiar puerto en flask_server.py:
```python
app.run(port=5001)  # Cambiar 5000 por otro número
```

### "Error de autenticación"
1. Eliminar users_db.json
2. Ejecutar setup_initial.py nuevamente
3. Usar credenciales de demo

### "Base de datos corrupta"
```bash
# Respaldar y eliminar
copy users_db.json users_db.json.backup
del users_db.json

# Reiniciar
python setup_initial.py
```

### "Aplicación lenta"
- Cerrar otras aplicaciones
- Actualizar Kivy: `pip install --upgrade kivy`
- Limpiar cache: eliminar carpeta `cache/`

## 📱 PANTALLA DE CONFIGURACIÓN AVANZADA

Desde la app, pestaña "Configuración" puedes:

1. **Viajes**
   - Establecer límites de distancia
   - Precio mínimo aceptable
   - Ratio mínimo de rentabilidad (RD$/km)

2. **Zonas**
   - Listar zonas favoritas
   - Agregar nuevas (nombre, lat, lon)
   - Listar zonas restringidas

3. **Interfaz**
   - Cambiar tema: Dark ↔ Light
   - Idioma: Español / Inglés

4. **GPS**
   - Mapas: Google o OpenStreetMap
   - Rango de visualización: 15/30/50 km o minutos

## 🚗 OVERLAY FLOTANTE DE VIAJES

Se muestra automáticamente cuando:
- Hay nuevo viaje disponible
- Aparece por 10 segundos
- Muestra: monto, distancia, duración, rating
- Botones: Aceptar / Rechazar

Para activar manualmente:
```python
from BOT_CORE.overlay_viajes import overlay_manager

overlay_manager.mostrar_viaje(
    monto=350.0,
    distancia=5.2,
    duracion_estimada=18,
    calificacion_pasajero=4.9,
    ubicacion_salida="Centro",
    ubicacion_llegada="Aeropuerto",
)
```

## 🎓 EJEMPLOS DE USO AVANZADO

### Crear zona automáticamente:
```python
from BOT_CORE.settings_manager import settings

settings.agregar_zona_favorita(
    "Mi Lugar Favorito",
    lat=18.486,
    lon=-69.931,
    radio_km=3.0
)
```

### Calificar usuario:
```python
from BOT_CORE.ratings_system import ratings_manager

ratings_manager.calificar_conductor(
    "driver_001",
    "trip_123",
    5,  # 1-5 estrellas
    "Excelente servicio"
)
```

### Obtener rutas:
```python
from BOT_CORE.gps_rutas import mapas_osm

ruta = mapas_osm.obtener_ruta(
    lat_origen=18.486,
    lon_origen=-69.931,
    lat_destino=18.591,
    lon_destino=-72.295
)

if ruta:
    print(f"Distancia: {ruta['distancia_km']} km")
    print(f"Duración: {ruta['duracion_minutos']} minutos")
```
